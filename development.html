Changelog V3-W0523

Modules

  • Added Block Progress module
  • Added Precision Sliders for modules such as Speed Counter or Reach Counter
  • Added Roundness slider to keystrokes
  • Added an option to disable Creative Tools TP lerp
  • Added Fast option for Creative Tools AutoClicker
  • Readded Audio Subtitles for 1.18.30+
    • It now has support for the new renderer, new features and various bug fixes. (New animation, display time, sound sorting, fixed width setting)
  • Removed useless Top Indicator in Toggle Sprint/Sneak settings
  • Rewrote Custom Crosshair
    • Removed click & load.
      • It will now load the crosshair as soon as you click the crosshair
    • UI has been revamped to use the V3 style
    • Added outline setting
    • Added crosshair previews

    Crosshair editor

    • Added pen width
    • Added fill tool
    • Added scroll to zoom
    • Added click and drag positioning
  • Added Only Display When Holding Bow/Crossbow to Arrow Counter
  • Added Don't Display If No Arrows to Arrow Counter
  • Added Added Don't Display If No Pots/Soups to Potion Counter
  • Added Always Center setting to Compass
  • Added a height offset to Light Overlay to combat snow layers
  • Added Day Counter
  • Added Low Health Warning
  • Added Particle Multiplier
  • Added Auto Text Hotkey
    • Sends a predefined message/command on a hotkey press
  • Added AutoGG again
    • Works on:
      • Hive
      • Hyperlands
      • Cubecraft
      • Zeqa
      • Nethergames
      • Galaxite

HUD

  • Revamped a LOT of UI
  • Added Scroll To Scale in the HUD editor
  • Removed the scaling arrow in HUD editor
  • Added Don’t Hide HUD In F1 setting to global settings
  • Rewrote the code for the scrollbar! Enjoy the new one :)
  • Added Theme presets
  • Added Chroma presets
  • Added Color presets to the color picker (left click to select, right click to apply new color)
  • Changed Notifications to be round
  • There is now a new default config
  • Added Outline to text modules

Bug fixes

  • Fixed a Keystrokes bug where only one input could be recognised
  • Fixed NBT savef functioning incorrectly if the item does not have a tag
  • Fixed TNT Timer not displaying anything
  • Fixed .give not functioning correctly on 1.16.40
  • Fixed 1.17.41 crashing on inject
  • Fixed a bug where resizing with the ClickGUI open would force you to Global Settings
  • Fixed Insert not opening the ClickGUI in when the HUD was hidden
  • Fixed Show Indicator not working in Toggle Sprint/Sneak settings
  • Fixed a bug where module toggles would function incorrectly when you right clicked and then clicked the toggle or vice versa
  • Fixed a bug where when leaving the UI, precision selected sliders would not unfocus if they were focused when you left
  • Fixed Combo Counter functioning incorrectly

Miscellaneous

  • Added Mouse Button 5 & 6 support to keybinds
  • Color Options no longer has an enabled/disabled button
  • Adapt Module Position now has priority loading
  • Added .directory command
  • Added Expand Direction setting for better module alignment
  • Added setting to Hide/Show “You are up to date!” when you inject
  • Improved the Freelook head-rotation glitch - can’t get you banned from servers anymore
  • Added Show Chat Message setting to Screenshot settings
  • Added Stop All Momentum key in Creative Tools
  • Added Perspective setting to Freelook
